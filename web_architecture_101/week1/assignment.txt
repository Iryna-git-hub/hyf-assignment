2. Identify the components:

Where is the recipe data stored? 
- Database

How does the frontend communicate with the backend? 
- Frontend sends requests to backend through API. 

What happens when a user adds a new recipe?
- User fills out a form and press "add recipe" button.
- Frontend sends POST request to Backend.
- Backend stores the new recipe in the database and return success response.
- Frontend updates the page.
